# NIST Method Pkg files

## **Content**
* Venus Methods files in the pkg format for the Semi Automated Passaging, Seeding and Maintenance, of the Allen cell collection of  hiPSC lines expressing green fluorescent protein tagged to protein identifying specific cellular organelles and structures. 
* You will find the instructions for the import in the word document in the folder.  The method files should be compatible with the latest version of Venus and associated drivers. If you need assistance when importing and opening due to needed driver and/or library please contact Ben Gregor at beng@alleninstitute.org or your local Hamilton applications person should be able to help you as well.
